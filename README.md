Индикатор разряда АКБ на ATmega8

Разработан индикатор критического разряда автомобильной батареи 12В. Напряжение АКБ через делитель R1/R2 (7.3кΩ:1кΩ) подается на AIN1 (PD7) ATmega8. Внутреннее опорное напряжение 1.3В компаратора служит пороговым значением на AIN0

При падении напряжения АКБ <11В сигнал на AIN1 становится <1.3В, ACO=1, красный светодиод DV2 загорается, зеленый светодиод DV1 гаснет. При нормальном напряжении ≥11В светодиод DV1 светится, а DV2 - гаснет.
